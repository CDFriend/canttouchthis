# canttouchthis
A secure instant-messaging application based on the Java Cryptographic
Architecture.

## Running from Source
The application consists of a client and server. The included Gradle wrapper
can be used to build and run either.

*Running the Client*
```
./gradlew run_client
```

*Running the Server*
```
./gradlew run_server
```

## Running Unit Tests
```
./gradlew test
```
